ReqLens
Analiza cualquier interfaz existente y obtén los requerimientos formales que necesitarías para construirla.
Qué obtienes al terminar
IEEE 830
Documento formal con requerimientos funcionales y no funcionales
Diagrama de módulos
Estructura visual de cómo se relacionan las partes del sistema
Modelo conceptual
Entidades y relaciones inferidas, sin diseño técnico de base de datos
Niveles de confianza
Cada dato marcado como confirmado, inferido o no accesible
Uso responsable
ReqLens analiza interfaces visibles públicamente con fines educativos y de aprendizaje.
No copia código, diseño visual ni contenido protegido — solo infiere requerimientos funcionales genéricos.
No debe usarse para clonar o competir directamente con el producto analizado.
Los resultados son sugerencias educativas, no especificaciones verificadas.
